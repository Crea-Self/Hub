Chaque point/tiret est dessiné à l'échelle réelle (points ronds, tirets rectangulaires) sur fond blanc. Le décodage relit ces formes comme un vrai signal Morse — fonctionne mieux sur une image non recompressée.

Déposez un fichier .wav / .mp3 contenant le signal (généré par cet outil), ou cliquez pour en choisir un.

Le Morse utilise une tonalité unique (bips courts/longs, comme un vrai émetteur). Les autres méthodes utilisent un « modem » à deux tonalités (comme un vieux modem téléphonique) : décodez toujours avec la même méthode qu'à l'encodage. Fonctionne mieux avec le .wav généré ici — le MP3 (compressé) peut parfois abîmer le signal.
